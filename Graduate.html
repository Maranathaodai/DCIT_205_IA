Course Descriptions

CSCD 603Advanced Data Structures and Algorithms This course will introduce students to a number of highly efficient algorithms and data structures for fundamental computational problems such as primality testing, linear optimisation and string matching. Other topics will include, design and analysis of algorithms, data types, trees, operations on sets, advance set representation methods, directed graphs, undirected graphs, sorting, algorithm analysis techniques, algorithm design techniques, data structures and algorithms for external storage as well as memory management. In addition students are introduced to randomised algorithms and to techniques of amortised complexity analysis.
